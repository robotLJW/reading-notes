# 大道--段永平

做对的事情，把事情做对。

## 第一章--投资大道

- 好的生意模式往往具有很宽的护城河。
- 看不懂就不碰。
- 错误是不可避免的，但待在能力圈内以及专注和用功可以大幅度减少犯错的机会。
- 我对信仰的理解就是“做对的事情”，做对的事情最后会有好的结果，或者说做错的事情最后会有大麻烦。
- 人生最重要的投资是投资在教育上。
- 投资使用的是闲钱，不然就是投机。
- 当你买一只股票时，你一定认为你在买这家公司，你可能拿在手里10年、20年。
- 投资像种田，投机像狩猎--狩猎是互为猎物的，是个零和游戏。
- 其实我也不愿意慢慢地变富，只是我不知道怎样才能快速变富而已。
- 人们最终会成为他自己，People never learn（人们从不吸取教训）。
- 看图看线实际上就是看目前（这个时刻或时段）别人对股票的看法，价值投资者的心中是无图无线的！
- 量化“投资”其实就是投机，是赚市场的钱，是零和游戏。
- 可惜很多人都是非要试过别的路才会醒悟过来的。
- 道需悟，术可学--道指得是做对的事情，或者说是不做不对的事情，或者说发现错了马上改，不管多大的代价都是最小的代价。术指的是技术，对每个人来说都是需要学同时也需要练习的，非一朝一夕。先有道然后术才有用，不然就是瞎忙。
- 道需要自己悟，大多数人都是不容易开放的，很难接受跟自己以前认识不同的东西。
- 不做空，不借钱，最重要的是不要做不懂的东西。
- 投资最重要的是要有独立思考的能力，要能够理性地思考。
- 假装会游泳是很危险的，假装会投资也一样。不过很多人不很理解假装会投资的危险在哪里，直到他们理解了危险。
- 把事情做对是要靠学习的，在实践中学习，在犯错中学习。不做永远都不会犯错，但是永远学不会。
- 最容易犯错的时候就是手里有很多现金的时候。
- 投资自己明白的生意，就不容易恐惧和贪婪。
- 平常心在这里指的是只想每一杆怎么打，别想结果，能做到的赢面就大了。
- 投自己懂的东西就容易知道价值，就知道什么时候叫便宜，就有机会赚钱，反之亦然。
- 投资最重要的是投在你真正懂的东西上。
- 坚持做正确的事，也就是原则性错误的事不做。
- 投资的一般规律是：出手越多，赚得越少，或赔得越多。
- 少犯错要通过坚持做对的事情来实现。而坚持做对的事情是发现错的事情就马上停止，不管多大的代价都会是最小的代价。不为清单也很重要。
- 总而言之，价值投资就是那种能睡好觉的投资。
- 没人希望自己需要富2次。
- 不要用margin是投资的基本要求！
- 把事情做对的能力只有在做对的事情时才能让时间站在自己的一边，不然早晚会出事的。
- 能从别人的错误里吸取教训的那都是天才。
- 估值是需要“功夫”的--时间。
- 想公司，想生意，想10年、20年后的公司是什么。
- 长线持有一定是在买对好股票的前提下，有些人往往只是在“被套”的时候才想起“长线”来。
- 时间是平庸公司的敌人，是伟大公司的朋友。
- 买自己一眼就看懂的生意往往都不错。
- 追高不择时的人们实际上对短期的下跌时非常难以忍受的。
- 做对的事情的人不一定具备把事情做对的能力，把事情做对需要很多年的幸苦积累，不是看一两本书或者上几个高手的博客就能学会的。
- 散户还是遵循低估买入，高估卖出比较合适。
